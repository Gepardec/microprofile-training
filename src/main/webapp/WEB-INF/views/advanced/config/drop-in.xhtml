MicroProfile-Config Advanced DropIn Config

The specification for MicroProfile Config (ConfigSource).

A custom ConfigSource can be implemented in its own JAR in order to maintain your configuration separately from your application.

Your Task:

Take a look at the module config/drop-in-configuration. There, implement and register a custom ConfigSource that retrieves the configuration from com.gepardec.training.microprofile.dropinconfiguration.DropInConfiguration. Finally, provide the drop-in-configuration JAR so that the configuration is present in this application:

  1. build the drop-in-configuration JAR (mvn clean install)
  2. add the drop-in-configuration-dependency to the pom.xml of the main application
  3. build the main application (mvn clean install)
  4. deploy the main application
