Config Advanced (Custom ConfigSource)

Configuration from custom ConfigSource

The specification for MicroProfile Config (ConfigSource).

Configuration can be provided by property file, environment variable, system property or a custom ConfigSource.

A custom ConfigSource allows individual retrieval of your configuration, for example from a database. Custom ConfigSources are treated as PoJo's and therefore can neither inject CDI beans nor can they use other MicroProfile ConfigSources.

Your Task:

Implement and register a custom ConfigSource that retrieves the configuration from the database. Use com.gepardec.training.microprofile.advanced.config.PropertyDao to access the database. Take a look at the controller class com.gepardec.training.microprofile.advanced.config.CustomConfigSourceController
