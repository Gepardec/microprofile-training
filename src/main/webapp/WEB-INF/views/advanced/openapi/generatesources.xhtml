The specification for Microprofile OpenAPI.

There are serveral ways to use the OpenAPI Generator e.g CLI, online generation and plugins. For our purpose the maven plugin openapi-generator-maven-plugin is the best pick.

Your Task:

Take a look at the OpenAPITools readme. Your goal is to configure the plugin in a way to achieve the following:

  • src folder src/gen/java/main
  • pojos generated in the folder model
  • uses microprofile as library
  • no pre generated tests
