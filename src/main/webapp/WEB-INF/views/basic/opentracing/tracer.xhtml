Open-Tracing Basic (Basics)

Open-Tracing Tracer

Instead of using the provided instrumentation implementations, we can provide our own spans by integrating our implementtions into JAX-RS lifecycle for instance.

We will provide spans for all of the JAX-RS requests to the training backend.

Your task:

Go to com.gepardec.training.microprofile.basic.opentracing.RestRequestFilter#filter and com.gepardec.training.microprofile.basic.opentracing.RestResponseFilter#filter method and use the injected tracer to create a span with the operation name <HTTP_PATH (request | response)> which contains the following tags:

  1. http-uri is the uri the span is for
  2. http-method the http method of the request
  3. http-header-<NAME> all of the set headers of the request
