Microprofile-Metrics Basics @Gauge

The specification for Microprofile Metrics.

@Gauge is an annotation for marking a method as a gauge. No default MetricUnit is supplied, so the unit must always be specified explicitly.

In this example we want to monitor the value of RandomNumberGenerator#getValue(). Therefore, use the @Gauge annotation. The name of the metric should be "gauge-example". Make sure that the metric is displayed at Wildfly and Prometheus.
