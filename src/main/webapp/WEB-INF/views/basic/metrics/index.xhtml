Microprofile Metrics Basic

With this example, you will use a basic metrics. Base Metrics

Use @Counted to measure how often an endpoint is invocated.

Use @Metric to retrieve measured values.

With @Metered annotation counts the invocations of the constructor or method and tracks how frequently they are called.

If you want to track and sample the duration for a method call, you can make use of timers ... for example @Timed annotation.
