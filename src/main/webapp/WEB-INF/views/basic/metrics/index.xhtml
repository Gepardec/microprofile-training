Microprofile Metrics Basic

Take a look at the already existing metrics.

Use @Counted to measure how often an endpoint is invocated.

Use @Metric to retrieve measured values.

With @Metered the invocations of the constructor or method are counted and the frequency of the calls is tracked.

If you want to track and sample the duration for a method call, you can make use of the @Timed annotation.

By using @SimplyTimed you are able to measure the duration and number of invocations.

With @Gauge you are able to retrieve the current return value of an annotated method.

Grafana allows you to visualize the metrics provided by our application. In this example, you will have a look at the Grafana Dashboard and add metrics to the dashboard.
