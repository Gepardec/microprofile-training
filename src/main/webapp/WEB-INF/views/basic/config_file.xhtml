The specification for Microprofile Config

Configuration can be provided by property file, environment variable, system property or a custom ConfigSource.

A property file is packed into the binary. Changes in the property file need repackaging of the whole application.

Let's take a look at the property file at src/main/resources/META-INF/microprofile-config.properties and the Controller class at com.gepardec.training.microprofile.basic.config.MPConfigController
