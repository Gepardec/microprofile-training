Enable telemetry

The specification for MicroProfile Telemetry (Enabling OpenTelemetry support).

OpenTelemetry support is disabled by default. You can read the full explaination why this behaviour wos chosen in the spec (link above).

Your Task:

First, check Grafana for any existing observability data. At this point, you shouldn't see any logs, traces, or metrics.

Next, enable OpenTelemetry support by adding the necessary configuration property.

Finally, take another look at Grafana. You should now see logs, traces, and metrics. To view even more data, browse through the different pages of this training and then check Grafana again.
