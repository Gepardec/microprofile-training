RestClient Basic (ExceptionMapper)

ExceptionMapper

The specification for MicroProfile RestClient (ExceptionMapper)

The MicroProfile RestClient does not out of the box map error responses to Exceptions so we have to register an ExceptionMapper on our own.

Your Task:

Implement com.gepardec.training.microprofile.basic.restclient.RestClientExceptionMapper to map the returned http-status code to a RuntimeException in case the http-status > 400.
Register the implemented exception mapper on the com.gepardec.training.microprofile.basic.restclient.HttpbinClientApi interface.
