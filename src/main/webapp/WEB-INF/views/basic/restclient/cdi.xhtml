RestClient with CDI

The specification for Microprofile Rest Client

When using CDI with MP-restclient the annotations @RestClient and @RegisterRestClient are needed. Also the baseUri has to be defined. This can be done direct as an annotation parameter for RegisterRestClient or in the MP-config file. The configKey is also an annotation parameter and it sets the prefix for MicroProfile-Config provided configurations. (Otherwise it would use the fully qualified class name)
