RestClient

In the first two examples you will learn how to access the microprofile-training/src/main/java/com/gepardec/training/microprofile/basic/restclient/CheetahService with MP. With JAX-RS it would probably look something like this:
Client client = ClientBuilder.newClient(); String response = client.target("http://localhost:8081/cheetahs").request().get(String.class);
With MP we can choose between CDI and the builder approach. Try to fix the errors and solve the tasks.
