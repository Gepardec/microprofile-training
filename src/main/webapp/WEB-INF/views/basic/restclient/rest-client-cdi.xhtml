The specification for MicroProfile RestClient (CDI)

When using CDI with MicroProfile RestClient the annotations @RestClient and @RegisterRestClient are needed. Also the baseUri has to be defined. The uri can be defined directly in the @RegisterRestClient annotation or provided via MicroProfile-Config.
