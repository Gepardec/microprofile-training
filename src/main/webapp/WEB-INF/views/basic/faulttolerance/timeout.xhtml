Timeout

The specification for Microprofile Fault Tolerance (Timeout)

Timeout limit method invocation durations and terminate them with a TimeoutException. Timeout behaviour prevents the service to become unresponsive if a method invocation hungs or takes longer as expected. The service is meant to skip the invocation in case of a hung or unexpected long run method invocation to keep the service responsive for other requests.
