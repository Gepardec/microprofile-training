The specification for Microprofile Fault Tolerance (Fallback)

Fallbacks are used to provide a default result for the caller, which actually can disguise the error from the caller. The caller doesn't necessarily knows that the result is a default result. Of course the fallback must make sense, and canot be used for all method invocations.
