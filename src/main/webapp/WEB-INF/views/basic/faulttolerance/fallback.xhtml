The specification for MicroProfile Fault Tolerance (Fallback)

Fallbacks are used to provide a fallback result for failing method invocations which the caller can work with. Of course, the fallback must match the result type of the failed method invocation and should provide a meaningful result for the caller. Fallbacks can be defined globally of for different exception types.
