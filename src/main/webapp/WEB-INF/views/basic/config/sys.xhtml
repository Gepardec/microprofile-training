Microprofile-Config System-Property Config

Configuration from system properties

The specification for Microprofile Config (Default ConfigSources).

Configuration can be provided by property file, environment variable, system property or a custom ConfigSource.

System properties are set outside of the application and therefore the application does not need to be rebuilt after changing the configuration. System properties can be altered at runtime, a system restart is not required.
