Microprofile-Config Basics Env. Config

The specification for Microprofile Config.

Configuration can be provided by property file, environment variable, system property or a custom ConfigSource.

Environment variables are set in the operating system and therefore the application does not need to be rebuilt after changing the configuration. However, the application needs a restart after altering environment variables.
