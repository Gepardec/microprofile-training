Microprofile-Config Basics File Config

Configuration from file

The specification for Microprofile Config.

Configuration can be provided by property file, environment variable, system property or a custom ConfigSource.

A property file contains key value pairs, for example key=value. It is packed into the binary, hence changes in the property file need repackaging of the whole application.
