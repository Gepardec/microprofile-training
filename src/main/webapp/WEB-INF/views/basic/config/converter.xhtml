Config Basic (Converter)

Configuration Converter

The specification for MicroProfile Config (Converter).

Converters allow the conversion of the configuration value from String into a desired targettype. By default, converters for common classes are provided by the MicroProfile Config, such as Boolean, Integer,... . A custom converter can be implemented for a specific target type.

Converters are treated as PoJo's and therefore can not have dependencies (e.g. CDI injections). If no converter is found, an automatic conversion is attempted.
