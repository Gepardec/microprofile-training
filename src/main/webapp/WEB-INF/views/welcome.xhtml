Training Microprofile done right

Welcome to the Microprofile done right training, where you will learn the several Microprofile specifications and how to use them properly. The following Microprofile specifications are covered by this training.

  • Microprofile-Config
  • Microprofile-Health
  • Microprofile-Rest-Client
  • Microprofile-OpenAPI
  • Microprofile-Metrics

The training is split into a Basic part where you are introduced to the several Microprofile specifications, and a Advanced part where you will lear how the Microprofile specifications interact with each other and how to tackle complex use cases with the learned mechanisms of the Microprofiles specifications.
